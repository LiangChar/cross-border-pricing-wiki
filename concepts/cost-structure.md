# 跨境电商成本结构

## 显性成本

### 采购成本
- 产品单价
- 包装材料费
- 仓储费

### 物流成本
- **头程**：工厂/供应商 → 海外仓
- **尾程**：海外仓 → 消费者
- 国内运费

### 平台成本
- 平台佣金（普遍 15% 左右）
- 支付手续费
- 广告营销费

## 隐性成本（容易被忽略！）

| 成本项 | 说明 | 我们是否有数据 |
|--------|------|:---:|
| 退款成本 | 退货物流+商品损耗 | 需核算 |
| 仓租 | 长期仓储费 | ✅ 已知压力大 |
| 汇率波动 | 汇率变化导致的损失 | 需监控 |
| 关税 | 各国关税差异 | 按目的国 |
| 运营成本 | 人工、办公等固定支出 | 不计入单品 |

## 隐性成本对利润的侵蚀

```
实际利润 = 显性利润 - 退款成本 - 汇率损失 - 仓租超额
```

很多卖家只算显性成本，发现"账面上赚钱，实际上亏钱"——原因就是隐性成本被忽略。

## 汇率风险管理

```
汇率风险影响 = 销售收入 × [（原始汇率 - 当前汇率）÷ 原始汇率]
```

对多币种结算的跨境电商尤为重要。
