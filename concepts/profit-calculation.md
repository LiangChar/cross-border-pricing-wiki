# 跨境电商利润核算

## 核心指标

### 毛利润
```
毛利润 = 销售额 - 产品成本 - 物流费 - 平台佣金 - 广告费 - 仓储费 - 退款成本
```

### 净利润
```
净利润 = 毛利润 - 运营成本（人工、办公）- 税费
```

### 利润率
```
利润率 =（净利润 ÷ 销售额）× 100%
```

## ROI（投资回报率）

```
ROI =（利润 ÷ 总投资）× 100%
```

- 总投资含：采购、广告、物流等所有投入
- **行业合格线 ≥ 150%**
- 低于 150% 说明投入产出比不健康

## ACOS（广告销售成本）

```
ACOS =（广告费用 ÷ 广告销售额）× 100%
```

衡量广告投放效率的核心指标。越低越好。

## ROAS（广告支出回报率）

```
ROAS = 广告销售额 ÷ 广告费用
```
- 投 100 元广告赚 500 元 → ROAS = 5
- **行业优质线 ≥ 3**

## TACOS（总广告成本占比）

```
TACOS =（广告费用 ÷ 总销售额）× 100%
```

衡量广告占整体生意成本的比重，避免形成广告依赖。

## 与我们业务的关系

| 指标 | 可用数据 | 用途 |
|------|---------|------|
| 销售额 | 月总销量(G) × 售价 | 收入端 |
| 采购成本 | 采购价(E) × 销量 | 成本端 |
| ROI | 理论增长指标 | 决策参考 |
