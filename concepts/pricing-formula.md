# 跨境电商定价公式

## 核心定价公式

### 基础成本加成定价
```
销售价格 = 总成本 ×（1 + 目标利润率）
```

**总成本构成：**
- 产品成本（采购价）
- 物流费（头程+尾程）
- 仓储费
- 平台佣金
- 税费

^[raw/articles/10100-pricing-formula-manual.md]

### 考虑平台佣金的精确定价
```
最终售价 =（总成本 + 目标利润）÷ [1 -（平台佣金率 + 支付手续费率 + 营销费率）]
```

这个公式的关键在于**先覆盖所有隐性成本再算售价**，避免利润被佣金和手续费吃掉。

> 示例：平台佣金 15%、目标利润 25% 时，需要在成本价基础上大幅上浮才能保证实际利润。

^[raw/articles/10100-pricing-formula-manual.md]

### 独立站简化版
独立站没有平台佣金，公式简化为：
```
售价 = 总成本 ×（1 + 目标利润率）
```

## 与我们飞书数据源的对应

| 公式参数 | 飞书字段 | 来源 |
|----------|---------|------|
| 采购成本 | 采购价(E列) | 店铺活动表 |
| 仓储费 | 仓租（已知业务规则） | 库存统计表 |
| 平台佣金 | 按平台配置 | 需手动配置 |
| 目标利润 | 理论增长指标 | 每月调整 |

## 两步法快速定价

1. **无利润定价（保本价）**：计算所有成本总和
2. **有利润定价**：`最终定价 = 无利润定价 ÷（1 - 利润率）`
