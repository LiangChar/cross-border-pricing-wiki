# 跨境电商库存管理公式

## 库存周转率

```
库存周转率 = 销售成本 ÷ 平均库存
平均库存 =（期初库存 + 期末库存）÷ 2
```

### 周转率与业务含义

| 周转率 | 含义 | 建议 |
|--------|------|------|
| >12 | 周转快 | 健康 |
| 6-12 | 正常 | 监控 |
| <6 | 周转慢 | 关注 |
| >28周未周转 | 滞销 | **必须清仓** |

> ⚠️ 超过 28 周（约 196 天）周转的产品需清货，避免长期仓储费（仓租）吃掉利润。

## 补货点计算

```
补货点 = 日均销量 × 交货周期 + 安全库存
```

- 日均销量 → 飞书 `近7日平均总销量(I)` 或 `月总销量(G) ÷ 30`
- 交货周期 → 供应商到仓时间（天）
- 安全库存 → 预防断货的缓冲量

## 最低库存水平

```
最低库存水平 = 安全库存 +（日均销量 × 补货周期）
```

## 库存资金占用率

```
库存资金占用率 =（库存成本 ÷ 总运营资金）× 100%
```

## 与我们飞书数据的对应

| 公式参数 | 飞书字段 | 列 |
|----------|---------|---|
| 在售库存 | 在售库存 | J |
| 月总销量 | 月总销量 | G |
| 近7日销量 | 近7日总销量 | H |
| 日均销量 | 近7日平均总销量 | I |
| 动销天数 | 动销天数 | K |
| 最长库龄 | 最长库龄(天) | L |

### 基于库龄的清仓决策

根据[[库存管理策略]]：
- 库龄 > 90 天且无在途 → **加速清仓**（仓租压力大）
- 库龄 > 196 天 → **必须清仓**（行业标准 28 周上限）
