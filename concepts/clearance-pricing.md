# 清仓定价策略（已验证版）

> ⚠️ 2026-05-26 重大修正：仓租不参与定价，只用于退仓决策。

## 核心原则

1. **仓租是沉没成本**，不会因为降价而消失。仓租大≠应该降更多价。
2. **降价唤醒滞销品无效**（97.7%数据验证）。零销量的品降了也卖不动。
3. **定价由库龄+销量驱动**，仓租只用于判断是否退仓止损。

|------|:---:|------|

| 库龄>196天 + 零销 | 50% | 成本价出清 |

| 动销>100天 + 零销 | 12% | TK秒杀 |
| 动销>60天 + 有销 | 15% | TK折扣 |
| 动销>60天 + 零销 | 8% | TK折扣 |
| 动销>30天 + 有销 | 10% | 满减 |
| 动销>30天 + 零销 | 5% | 满减 |

## 不需要定价的情况

- **仓租>200/月 且 近7零销量** → 退仓止损，不定价

## 定价后还需检查

- 仓租>200/月 且 估月盈亏<0 → 决策建议"退仓止损"（价格给了，但不建议继续卖）

## 平台规则约束

### TK秒杀
- 有30天订单: 秒杀价 ≤ 近7折后最低价（取95%）
- 无订单: 秒杀价 ≤ 原零售价

### TK折扣
- 折扣价 ≤ 折后均价 × 85%

### 底线
- 最低售价 ≥ 79 MXN

## 到手价公式（TK）

```
到手价 = 活动标价 × 0.895 × 0.88 - 18(操作费)
利润 = 到手价 - 成本(不含操作费)
```
